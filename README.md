## REST API Template
Build REST APIs using this template for the project.
This project uses MVC structure.

## Features
1. Mysql 2 (connection, query api (prepared statements))
2. Express with router
3. MVC
4. Dotenv
5. JWT Auth set up
6. MiddleWares
7. Logging (wisnton)
